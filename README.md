## Website Name:ContestHub
## live Link: https://golden-dusk-c8a239.netlify.app/

## Website features:
- user can search contest by contest type in the search bar.
- user can login and sign up using google login and email password
- user can participate in the contests
- user can payment in the contest payment option
- user can see statistics page
- user can learn from resources
- there has different type of dashboard for different role  like admin, user, creators
- creator can post, update contest and declare winner
- admin can change user role and approved post
- user can see leaderboard, wining contest, participated contest. and can update their profile.
